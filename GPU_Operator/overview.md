# GPU Operator Overview

## Mục đích

Thư mục `GPU_Operator` được dùng để tổng hợp thông tin và tài liệu về việc vận hành GPU Operator.

## Gợi ý mở rộng

- Kiến trúc và các thành phần chính
- Cách cài đặt trên Kubernetes
- Theo dõi tình trạng GPU
- Xử lý sự cố liên quan đến driver, toolkit và plugin
