# MLflow Notebooks

Thư mục này hiện được tách thành 3 notebook theo đúng phạm vi sử dụng:

- `mlflow_ML.ipynb`: MLflow cho Machine Learning truyền thống, benchmark nhiều thuật toán, nested runs, model artifact.
- `mlflow_DeepLearning_HuggingFace.ipynb`: MLflow cho PyTorch, Deep Learning, Hugging Face và Transformer.
- `mlflow_LLM_Agent.ipynb`: MLflow cho LLM application, Prompt Registry, tracing, evaluation và AI Agent.

Gợi ý thứ tự đọc:

1. Bắt đầu với `mlflow_ML.ipynb` nếu muốn nắm nền tảng tracking cơ bản.
2. Đọc `mlflow_DeepLearning_HuggingFace.ipynb` khi chuyển sang workflow huấn luyện theo epoch hoặc Transformer.
3. Đọc `mlflow_LLM_Agent.ipynb` khi làm ứng dụng GenAI, prompt engineering, RAG hoặc agent.
